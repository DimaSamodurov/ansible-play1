# Creating sample Dynamic Inventory and a plugin for Ansible

## Configuration
Sample uses your local AWS configuration, e.g. `~/.aws/config` and `~/.aws/credentials`

## Inventory usage

    # To display the help
    ./ec2hosts.py
    # To describe aws instances
    ./ec2hosts.py --list
